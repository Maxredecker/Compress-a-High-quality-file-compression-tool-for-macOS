Formats

  • JPG / PNG / AVIF → JPG, WebP, AVIF, JXL
  • SVG → optimized SVG (SVGO)
  • PDF → compressed PDF (Ghostscript)

Optional tools

For JXL output: brew install jpeg-xl
For PDF compression: brew install ghostscript
